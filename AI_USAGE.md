# AI Usage Declaration

## Project
**Consultation Recording Manager**

## AI Tools Used
The following AI tools were used during the development process:

- ChatGPT (OpenAI)
- Bolt.new (for generating and refining UI components and layouts)
- v0 (for UI ideation and design exploration)

## Purpose of AI Usage

AI tools were used as development assistants to improve productivity and accelerate the implementation process. Specifically, AI was used for:

- Brainstorming and refining project ideas.
- Creating the initial Product Requirements Document (PRD).
- Generating frontend UI concepts and layouts.
- Assisting with responsive design improvements.
- Providing guidance on project architecture and folder structure.
- Suggesting code snippets for common patterns such as authentication, API integration, and component organization.
- Helping draft project documentation, README content, and future scope discussions.

## Human Contribution

All major technical decisions, implementation choices, debugging activities, testing, and final code integration were performed manually by the project author.

The author was responsible for:

- Designing the final project scope and feature prioritization.
- Setting up and configuring the MERN stack environment.
- Implementing and validating backend APIs.
- Integrating frontend components with backend services.
- Managing MongoDB schemas and database relationships.
- Debugging authentication, routing, file uploads, and search functionality.
- Testing end-to-end workflows to ensure correctness.
- Reviewing, modifying, and adapting AI-generated suggestions to fit project requirements.

## Verification Statement

Any AI-generated code or recommendations were carefully reviewed, tested, and modified before inclusion in the final project. The final submission represents the author's understanding of the implemented features and architecture.
